CABLE CONECTOR DE LA FUENTE DE UN PC

Primero de todo os presentamos el tipo de cables más antiguamente usado y que actualmente se relega a las fuentes más baratas. Este consiste simplemente en unir un terminal con otro mediante un cable suelto de colores, teniendo dos particularidades especiales: El color del cable denota el voltaje que irá por él, para facilitar diagnósticos y montajes. Estamos hablando de rojo para 5V, amarillo para 12V, negro para el negativo, naranja para 3.3V, morado para 5VSB, azul para -12V, y luego tenemos señales especiales como la POWER_GOOD gris o la PS-ON verde. Como decimos, no hay ninguna unión especial entre los cables para organizarlos mejor, más allá (como mucho) de una brida.
